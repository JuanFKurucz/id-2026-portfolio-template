# Recursos útiles

Usá esta página como punto de partida para trabajar en tu portafolio. No es una bibliografía completa: son los enlaces mínimos para no perderte entre consigna, evidencia y publicación.

## Curso y consignas

- Sitio de tareas de la materia: `../ucu-id-tareas/docs/index.md` si estás trabajando dentro del repo global del profesor, o el sitio publicado que indique el docente.
- Onboarding del portafolio: [onboarding-primera-semana.md](onboarding-primera-semana.md).
- Plantilla de entrada: [portfolio/plantilla.md](portfolio/plantilla.md).
- Evidencia, defensa y bonificación: [plantillas/evidencia-defensa-portfolio.md](plantillas/evidencia-defensa-portfolio.md).
- Declaración AI-safe: [plantillas/entrega-ai-safe.md](plantillas/entrega-ai-safe.md).

## Publicación

- MkDocs local: `mkdocs serve`.
- Build estricto: `mkdocs build --strict`.
- GitHub Pages: revisar la configuración del repo personal y dejar captura local si la publicación todavía no funciona.

## Criterio de uso

- Guardá en el portafolio solo evidencia seleccionada y defendible.
- No subas `.env`, tokens, datasets privados, correcciones ni datos de compañeros.
- Para WebAsignatura entregá solo el link o archivo pedido por el docente.
