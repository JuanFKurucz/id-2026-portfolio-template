# Índice del portafolio

Esta sección reúne las entradas que vas a defender durante el curso. Cada entrada debe mostrar una evidencia concreta, una decisión propia, un límite y una forma de reproducir o inspeccionar el resultado.

## Cómo crear una entrada nueva

1. Copiá [plantilla.md](plantilla.md) y guardala con nombre numerado, por ejemplo `02-eda.md`.
2. Completá solo lo que puedas defender.
3. Enlazá notebooks, scripts, capturas o resultados con rutas relativas.
4. Agregá declaración AI-safe cuando corresponda.

## Entradas iniciales

- [Mapa de evidencias](mapa-evidencias.md): checklist de 16 clases, parciales, ranking interno y defensa final.
- [Primera entrada](01-primera-entrada.md): orientación, arranque y primera evidencia del portafolio.
- [Plantilla](plantilla.md): estructura recomendada para nuevas entradas.

## Antes de entregar

- Revisá que no publicaste secretos, datos privados ni respuestas A0 completas.
- Verificá que la evidencia clave sea reproducible o inspeccionable.
- Prepará una microdefensa de 2-4 minutos.
- Entregá en WebAsignatura solo el link o archivo pedido por el docente.
